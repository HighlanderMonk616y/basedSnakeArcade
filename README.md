# Basecade 🐍

A fun little browser-based **Snake arcade game**, just like classic arcade cabinets.

Built commit-by-commit (100 commits) to level up GitHub contribution stats.

## 🎮 How to Play

1. Open `index.html` in your browser
2. Use **Arrow Keys** to control the snake
3. Eat the red food to grow and increase score
4. Don't hit the walls or yourself!

## ✨ Features (will be added over 100 commits)

- Classic Snake gameplay
- Retro arcade style
- Score tracking
- Game Over screen
- Start / Restart functionality
- Increasing speed
- High score saving (localStorage)
- Sound effects (planned)
- Mobile touch controls (planned)

## 🛠️ Tech Stack

- HTML5 Canvas
- Vanilla JavaScript
- Pure CSS for retro feel
